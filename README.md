# JiuChess AI - BetaJiu
民大-藏族久棋AI算法平台(MUC JiuChess AI - BetaJiu) 

1.技术及功能介绍：使用Python、Html、JavaScript等编程语言和Flask框架实现的藏族久棋算法实验平台。该平台具有人人对弈、机机对弈、人机对弈、保存棋谱数据、算法自对弈生成h5py数据集文件等功能。
